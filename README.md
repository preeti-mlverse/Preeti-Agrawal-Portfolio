# Preeti Agrawal - Digital Portfolio

A clean, illustrative digital portfolio web app for a senior Product Manager with expertise in AI, Analytics, and Automation.

## Live Preview

**Sandbox URL**: https://3000-iz4mitcuddnkythyhr6zl-ea026bf9.sandbox.novita.ai

## Project Overview

- **Name**: Preeti Agrawal Portfolio
- **Goal**: Showcase 10+ years of product leadership experience across e-commerce, D2C, SaaS, and fintech
- **Target Audience**: Product leaders, hiring managers, and founders at SaaS/FinTech/AI-enabled companies

## Features

### Completed Features
- **Horizontal Tab Navigation** - Easy navigation between About, Journey, Projects, Skills, Education, and Contact sections
- **Creative Timeline Visualization** - Alternating left-right timeline showing career progression from 2012 to present
- **Detailed Project Cards** - Each project includes Why, What, For Whom, How, and Impact/Metrics sections
- **Flat Editorial Illustrations** - SVG illustrations matching the calm, intelligent design system
- **Segoe Print Typography** - Handwritten-style font throughout for a personal, approachable feel
- **Responsive Design** - Works on desktop and mobile devices
- **Smooth Animations** - Fade-in effects and hover interactions

### Design System
- **Color Palette**: Soft, muted colors (indigo, lavender, coral, sage, peach, mint, sky)
- **Typography**: Segoe Print, Bradley Hand, cursive fallbacks
- **Illustration Style**: Flat/semi-flat editorial, minimal facial detail, rounded shapes
- **Layout**: Card-based structure with ample whitespace

## Sections

1. **About** - Hero section with quick stats and "What I Do" cards
2. **Journey** - Interactive timeline of career progression
3. **Projects** - 7 detailed project case studies with metrics
4. **Skills** - Categorized skill tags (Product, AI/Tech, Technical, Tools)
5. **Education** - Academic credentials and certifications
6. **Contact** - Contact information and social links

## Tech Stack

- **Framework**: Hono (Cloudflare Pages template)
- **Styling**: TailwindCSS (CDN)
- **Icons**: Font Awesome
- **Build Tool**: Vite
- **Runtime**: Cloudflare Workers

## API Endpoints

- `GET /` - Returns the portfolio HTML
- `GET /api/profile` - Returns profile JSON data

## Local Development

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Start development server
npm run dev

# Or use PM2 (sandbox environment)
pm2 start ecosystem.config.cjs
```
